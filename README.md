# 🍕 Pizza Restaurant API

This is a RESTful API for a fictional pizza restaurant built with **Flask**, following the **MVC** architecture. It allows you to manage restaurants, pizzas, and the relationship between them.

---

## 🛠️ Technologies Used

- Python 3
- Flask
- SQLAlchemy
- Flask-Migrate
- Pipenv
- SQLite (development)
